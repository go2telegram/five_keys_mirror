# Задача
Собери план на 7 дней: **утро / день / вечер**. Для каждого блока:
- 1–2 действия образа жизни (сон/вода/движение/стресс-менеджмент)
- 1 нутри-подсказка (если уместно) с **конкретикой** из каталога:
  - название товара, как принимать, когда, на сколько (курсом)
  - почему (на основе тегов/результатов: "по тесту сон низкий → магний вечером", "энергия ↓ → MCT утром")

# Ограничения
